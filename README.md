# Co-processador-N-cleo-ELM-Verilog-Marco-1-
Projeto em Verilog para a placa DE1-SoC que implementa um co-processador baseado em ELM para classificação de dígitos. O sistema inclui FSM de controle, memórias para imagem, pesos e bias, ativação em hardware e saída da predição, além de suporte a testes em placa e futura integração com processador.
